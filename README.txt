# Secure Chat App with Guaranteed Backup

Passwords:
- bulbul (normal mode, messages expire after 24 hours)
- kakutis (show all messages including deleted, never expire)
Features:
- Persistent storage with guaranteed backup
- Russian labels
- Popup modal after login
- Messages with HH:MM format
- Delete message option (Удалить)
- Send images along with text
- Re-authentication every 3 minutes
- Show number of connected users
